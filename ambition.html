🚀 Ambition Engine

📌 Overview

Ambition Engine is a cloud-based task management system designed to manage daily goals efficiently. It allows users to create, update and track tasks in real time.

⚙️ Tech Stack

  • Java
  • Spring Boot
  • Hibernate / JPA
  • MySQL
  • Railway (Deployment)

🔥 Features

  • User Authentication
  • CRUD Operations for Tasks
  • REST APIs
  • Cloud Deployment
