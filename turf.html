⚽ TurfMatch

📌 Overview

TurfMatch is a sports matchmaking platform where users can host or join games. It helps players find nearby matches and manage bookings easily.

⚙️ Tech Stack

  • Java
  • Spring Boot
  • Hibernate
  • MySQL

🔥 Features

  • Host a Match
  • Join Games
  • Player Management
  • Slot Booking System

🔗 Project Idea

Built to solve real-world problem of finding players locally for sports like cricket and football.
